# 📊 Mobile Sales Analytics Dashboard

## Overview
This project is an interactive Power BI dashboard designed to analyze mobile sales performance, customer behavior, and business trends.

## Features
- Total Sales Analysis
- Total Quantity Sold
- Transaction Analysis
- Average Sales Tracking
- City-wise Sales Visualization
- Monthly Sales Trends
- Customer Ratings Analysis
- Payment Method Distribution
- Mobile Brand Performance
- Mobile Model Performance

## Tools & Technologies
- Power BI
- Data Analytics
- Data Visualization
- Business Intelligence
